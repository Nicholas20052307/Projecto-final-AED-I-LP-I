# Projecto-final-AED-I-LP-I

## Descrição
Este projecto implementa um sistema de análise de texto em C, baseado em tokenização,
vetorização TF e cálculo de similaridade entre frases, utilizando estruturas de dados dinâmicas.

## Estrutura do Projecto
- src/ – código fonte
- tests/ – funções de teste dos requisitos
- README.md – descrição do projecto

## Requisitos Implementados

| Requisito | Estado | Observações |
| R1.1 | Implementado | Matrizes dinâmicas de strings |
| R1.2 | Implementado | Inserção, remoção e pesquisa |
| R1.3 | Implementado | Fusão iterativa de tokens (BPE simplificado) |
| R1.4 | Implementado | Tokenização greedy longest-match |
| R1.5 | Implementado | Vetorização TF |
| R1.6 | Implementado | Produto interno e cosseno |
| R2.1 | Implementado | Lista ligada LL_TK_TF |
| R2.2 | Implementado | Estruturas MATRIX_STR, MATRIX_INT e DOC |
| R2.3 | Implementado | Ficheiros de texto |
| R2.4 | Implementado | Ficheiros binários |
| R2.5 | Implementado | Documentação e Git |

## Compilação
O projecto foi desenvolvido em linguagem C, utilizando memória dinâmica e modularização.
